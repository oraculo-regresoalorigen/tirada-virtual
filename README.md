🔮 Oráculo Virtual: Regreso al Origen - Conexión Cristal
Bienvenido al repositorio oficial del Oráculo Virtual, una experiencia digital inmersiva diseñada para la marca Conexión Cristal. Este proyecto fusiona tecnología moderna con sabiduría ancestral, permitiendo a los usuarios interactuar con las 21 llaves del origen a través de una interfaz de lujo (Luxury UI).
✨ Propuesta de Valor
Este oráculo no es solo una herramienta de consulta; es un Lead Magnet de alta conversión diseñado para:
Generar engagement profundo con la audiencia.
Reforzar la autoridad de la marca en el nicho de espiritualidad y soberanía biológica.
Ofrecer una experiencia visual "Premium" que se adapta a cualquier dispositivo.
🚀 Características Técnicas
Frontend: HTML5 semántico y CSS3 con animaciones avanzadas.
Styling: Tailwind CSS para un diseño responsivo y moderno.
Interactividad: JavaScript Vanilla (sin dependencias externas) para garantizar velocidad de carga y fluidez.
Efectos Visuales:
3D Card Flip: Perspectiva de 2000px para un realismo táctil.
Floating Animations: Micro-interacciones que dan vida a la interfaz.
Glassmorphism: Uso de filtros de desenfoque y bordes dorados para una estética "Cristal".
Tipografía: Integración de Cormorant Garamond (elegancia clásica) y Montserrat (modernidad limpia) vía Google Fonts.
🛠️ Estructura de Contenido
El oráculo gestiona una base de datos interna de 21 Llaves, cada una con:
Mensaje Principal: El núcleo de la enseñanza.
La Sombra: El aspecto a observar o integrar.
La Integración: Acción sugerida para la coherencia biológica.
Pregunta para el Alma: El detonante de la auto-reflexión.
📱 Optimización UX/UI
Mobile First: Totalmente optimizado para navegadores de smartphones.
Accesibilidad: Contraste cuidado y fuentes legibles.
Feedback Visual: Botones con efectos de gradiente dorado y estados de hover refinados.
